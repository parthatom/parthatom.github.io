Learning PCA? SVM? K-Means Clustering? Ask yourself these questions for a thorough understanding of these algorithm.

An (inexhaustive) list of questions to ask when learning about a machine learning algorithm

  1. What does the algorithm do?
    2. Try describing the algorithm to a 5 year old. Avoid as much terminology as you can, any presumptive knowledge and all the "big" words. Will you be able to explain it?

  2. What type of data does the algorithm work on?

    3. Does it work on N-dimensional vectors or images? If it works for categorical variables can be it used for images, what about sequential or time series data? Is the source of the data important?
